# K-LOG UI/UX Overhaul Plan

## 1. 개요
현재의 실용적인 기능을 바탕으로, 트레이딩 전문가용 플랫폼 느낌이 나는 **프리미엄 다크 테마(Premium Dark UI)**로 전체 디자인을 개편합니다.

## 2. 디자인 키워드
- **Glassmorphism**: 은은한 투명도와 블러 효과를 사용한 카드 디자인
- **Neon Accents**: 중요한 수치와 액션 버튼에 세련된 네온 컬러 포인트
- **Micro-interactions**: 부드러운 전환 효과와 호버 애니메이션
- **Typography**: 정보 계층 구조가 명확한 폰트 시스템

## 3. 주요 개선 사항

### A. 컬러 시스템 및 배경 (Visual Base)
- **Deep Space Black**: `#05070a` -> `#020617` (깊이감 있는 네이비 블랙)
- **Glass Card**: 배경 투명도 50~70% + `backdrop-filter: blur(12px)`
- **Accent Color**: 기존 네온 블루(`--accent`)를 유지하되, 그라데이션을 더 적극적으로 활용

### B. 내비게이션 및 헤더 (Navigation)
- 헤더 높이 및 패딩 최적화
- 메뉴 호버 시 아래쪽 바가 생기는 대신, 배경이 은은하게 밝아지는 글래스 효과 추가
- 로고 폰트 무게감 상향 및 빛나는 효과(Glow) 추가

### C. 컴포넌트 고도화 (Components)
- **Pro-Card**: 
    - 테두리를 더 얇고 반투명하게 변경 (`rgba(255,255,255,0.05)`)
    - 카드 내부 섹션 간 구분선 제거 또는 최소화
- **Tables**:
    - 셀 패딩 최적화
    - 행(Row) 사이의 보더 대신 아주 옅은 배경색 차이(Zebra stripe) 활용
- **Buttons & Inputs**:
    - 모든 입력창에 포커스 시 네온 글로우 효과 추가
    - 버튼에 미세한 그라데이션 및 입체감 부여

### D. 레이아웃 최적화 (Layout)
- 3단 그리드 간격(`gap`)을 24px로 확대하여 여백의 미(White space) 확보
- 사이드바 내의 카드 배치 순서 및 중요도 재조정

## 4. 기술적 구현 계획
1.  **CSS 변수 재설정**: `:root`에서 색상 및 그림자 변수 고도화
2.  **전역 스타일 업데이트**: 폰트 사이즈, 기본 마진, 스크롤바 스타일 개선
3.  **컴포넌트 스타일 리팩토링**: 헤더, 카드, 테이블 스타일을 순차적으로 교체
4.  **반응형 대응**: 미디어 쿼리를 사용해 태블릿 및 모바일 가독성 개선

## 5. 예상 결과
- 사용자가 사이트에 접속했을 때 "전문적인 분석 도구"를 사용하고 있다는 강력한 시각적 신뢰감 부여
- 장시간 사용 시 눈의 피로도를 낮추는 차분하면서도 선명한 대비율 확보
